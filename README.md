# ⚒️ SchemaForge

> Intelligent Database Schema Generator for AI Agents.

SchemaForge is a developer tool that simplifies database modeling for autonomous agents and rapid MVP development. It generates clean, optimized SQL and Type definitions based on high-level data descriptions.

## ✨ Features
- 🏗️ **Smart Scaffolding:** Generate Postgres/Supabase tables in seconds.
- 📜 **Auto-Types:** Instant TypeScript interface generation from schema.
- ⚡ **Vite-Powered:** Built for speed and modern developer experience.
- ☁️ **Cloud Ready:** Optimized for Supabase integration.

## 🛠️ Tech Stack
- **Framework:** Vue 3 + Nuxt 3
- **Styling:** Vanilla CSS Variables
- **Deployment:** Vercel
